使用Hexo搭建个人博客，Hexo 是一款基于 Node.js 的静态博客框架。

并部署到github pages。

一、本地搭建

（此阶段只需node.js和hexo即可）

  1. 命令行安装hexo框架：npm install -g hexo-cli hexo

​ 验证是否安装成功： hexo -v

  1. cd一个位置并执行：hexo init 项目名 //生成 Hexo 项目骨架和依赖清单

​ 创建成功后：cd 项目名 //进入项目目录

  1. 读取package.json依赖清单，下载所需依赖包到node_modules: npm install

  2. 清理、生成静态文件并启动本地服务：hexo clean && hexo g && hexo s

    浏览器访问 http://localhost:4000/ 测试效果 CTRL+C 终止服务

  • 在“_config.yml”文件中配置hexo：官方文档https://hexo.io/zh-cn/docs/configuration

  • 安装butterfly主题：官方网站：https://butterfly.js.org/posts/21cfbf15/

    安装命令：git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

    复制./themes/butterfly/_config.yml 文件，到根目录文件夹下（和 _config.yml 同级），并修改文件名为：_config.butterfly.yml

    Hexo 会优先读取这个文件的配置，后续改主题样式、加功能都改这个文件。

    修改根目录下的 _config.yml文件，把主题(theme)改为 butterfly。

    安装 butterfly的渲染器：npm install hexo-renderer-pug hexo-renderer-stylus --save

二、云部署

  1. git配置：

​ git config --global user.name <你的用户名>

​ git config --global user.email <你的邮箱>

​ 查看是否配置成功：git config -l

  1. 使用SSH连接至 GitHub：

​ 运行：ssh-keygen -t rsa -C <你的邮箱>

​ 直接按几次回车生成ssh密钥，用于此电脑连接至 GitHub

​ 之后打开C盘用户文件夹下的.ssh的文件夹，用记事本打开 id_rsa.pub（公钥），复制里面的内容

​ 然后github点右上角头像 > Settings > SSH and GPG keys > New SSH key

​ Title随意，Key粘贴刚刚复制的密钥，之后点Add SSH key

​ 运行以下代码测试连接：ssh -T git@github.com //出现连接到账户即成功

  1. 安装 Git 部署依赖：npm install hexo-deployer-git --save

  2. 缓存清理、静态文件生成、远程推送: hexo cl && hexo g && hexo d
