  • 使用Hexo搭建个人博客。（Hexo 是一款基于 Node.js 的静态博客框架。）

  • 部署到github pages。（可通过 “https://<github用户名>.github.io” 访问）

一、本地搭建

此阶段仅node.js和hexo即可

1、安装hexo框架

  • 命令行执行：npm install -g hexo-cli

  • 验证是否安装成功： hexo -v

2、创建hexo项目

  • cd一个位置并执行：hexo init 项目名 //生成 Hexo 项目骨架
  • 创建成功后：cd 项目名 //进入项目目录
  • 自动读取package.json依赖清单，下载所需依赖包到node_modules: npm install

3、启动本地服务

  • 清理、生成静态文件并启动本地服务：hexo cl && hexo g && hexo s

  • 测试：http://localhost:4000/

  • 按CTRL+C 终止服务

三、网站美化与内容创作

网站配置

  • 在“_config.yml”文件中配置hexo：官方文档https://hexo.io/zh-cn/docs/configuration

主题

安装butterfly主题

  • 官方网站：https://butterfly.js.org/posts/21cfbf15/ （也可选择其他主题）

  • 安装命令：git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

  • 复制./themes/butterfly/_config.yml 文件，到根目录文件夹下（和 _config.yml 同级），并修改文件名为：_config.butterfly.yml

  • 主题配置：_config.butterfly.yml

    网站配置：_config.yml

  • 修改根目录下的 _config.yml文件，把主题(theme)改为 butterfly。

  • 安装 butterfly的渲染器：npm install hexo-renderer-pug hexo-renderer-stylus --save

文章

新增博客

  • 首先cd到博客根目录

  • hexo list post //查看所有文章

  • hexo new 文章名 //此命令将在source/_post目录自动生成 文章名.md文件
